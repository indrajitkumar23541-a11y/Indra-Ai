## 🌟 Introduction

**Indra Ai** is a powerful, highly-customizable platform that allows you to create, deploy, and manage continuous AI agents that automate complex workflows. Built independently by **Indrajit Kumar**, this platform represents the cutting-edge of autonomous AI task management.

Whether you want to automate your daily tasks, generate dynamic content, or build multi-step AI reasoning pipelines, Indra Ai provides the ultimate workspace to bring your automation ideas to life.

---

## 🚀 Key Features

### 🧱 Intuitive Frontend Interface
The Indra Ai frontend is where users interact with our powerful AI automation platform. It offers multiple ways to engage with and leverage our AI agents:

- **Agent Builder:** An intuitive, low-code interface allowing you to design and configure your own AI agents effortlessly.
- **Workflow Management:** Build, modify, and optimize your automation workflows with ease by connecting logical blocks.
- **Ready-to-Use Agents:** Select from a library of pre-configured agents and put them to work immediately.
- **Real-Time Monitoring:** Keep track of your agents' performance and gain insights to continually improve your processes.

### 💽 Robust Backend Server
The Indra Ai Server is the powerhouse of the platform. Once deployed, agents can be triggered by external sources and can operate continuously with maximum reliability.

- **Native Qwen Integration:** Direct support for Qwen's DashScope models (`qwen-max`, `qwen-plus`, `qwen-turbo`) out-of-the-box.
- **Source Code:** The core logic driving autonomous agents.
- **Scalable Infrastructure:** Designed to handle heavy workloads and multi-agent coordination.

## 🛠️ Getting Started (Self-Hosting)

> **Note:** Setting up and hosting the Indra Ai Platform is designed for developers and tech enthusiasts.

### System Requirements
- **CPU:** 4+ cores recommended
- **RAM:** Minimum 8GB (16GB recommended)
- **Software:** Docker Engine, Docker Compose, Git, and Node.js.

### Quick Setup

Clone the repository and run the setup using Docker Compose:

```bash
git clone https://github.com/indrajitkumar23541-a11y/Indra-Ai.git
cd Indra-Ai/autogpt_platform
docker compose up -d
```

Once the containers are running, navigate to `http://localhost:3000` in your browser to access the Indra Ai interface!
